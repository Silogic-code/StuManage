# 学生成绩管理系统 - 测试用例
# 使用方法：
#   1. 编译程序：bash run.sh build
#   2. 运行程序：bash run.sh run
#   3. 按每个用例的说明输入数据
#   4. 数据输入完毕后：
#      - Windows: Ctrl+Z 然后按 Enter
#      - Linux/Mac: Ctrl+D
#   5. 按交互指令操作

================================================================================

提示
================================================================================

- 所有测试用例的编号查找均为 0-based（0 ~ size-1）
- Windows 结束数据输入：Ctrl+Z 然后按 Enter
- Linux/Mac 结束数据输入：Ctrl+D
- 如果结束输入快捷键无效，随便输入一个字母，程序会自动结束录入
